RETROALIMENTACIÓN

Pregunta 1

¿Qué es el polimorfismo en programación orientada a objetos?

RESPUESTA:

C. Una técnica para tratar objetos de diferentes clases de manera uniforme.

Pregunta 2

¿Cuál de las siguientes palabras clave se usa en Java para permitir el polimorfismo?

RESPUESTA:

B. override

Pregunta 3

¿Cuál es el beneficio principal del polimorfismo en la programación?

RESPUESTA:

B. Facilita la reutilización del código
