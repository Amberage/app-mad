Encapsulamiento

Actividad 1

OBJETIVOS

  • 1. Comprender el concepto de encapsulamiento en la programación orientada a objetos.
  • 2. Identificar situaciones del mundo real en las que el encapsulamiento es esencial.
  • 3. Evaluar si un escenario dado requiere encapsulamiento o no.

ACTIVIDAD

A continuación, se presentan ejemplos de escenarios reales, lee y analiza si es esencial el encapsulamiento o no, dependiendo de la situación.

Escenario 1: Aplicación de Gestión de Correos Electrónicos

Problema: En una aplicación de gestión de correos electrónicos, es necesario rastrear la dirección de correo electrónico de un usuario.

De acuerdo con el problema planteado, ¿es necesario aplicar encapsulamiento a "correoElectronico"?

Escenario 2: Sistema de Registro de Eventos Públicos

Problema: En un sistema que registra eventos públicos, como conferencias o conciertos, es necesario rastrear la fecha de cada evento.

¿En relación con el problema planteado, se debe encapsular la variable "fechaEvento"?

Escenario 3: Sistema de Control de Inventario en una Tienda de Ropa

Problema: En un sistema de control de inventario en una tienda de ropa, es necesario llevar un registro de la cantidad de existencias de cada artículo de ropa.

¿Según el problema planteado, es necesario encapsular la variable "cantidadExistencias"?

Escenario 4: Sistema de Control de Temperatura en una Industria de Alimentos

Problema: En un sistema de control de temperatura en una fábrica de alimentos, es fundamental rastrear la temperatura de almacenamiento de un producto.

¿De acuerdo con el problema planteado, es necesario encapsular la variable "temperaturaAlmacenamiento"?

Escenario 5. Clase de Constantes Matemáticas

Problema: Suponga que tiene una clase donde las variables son constantes de matemáticas, por ejemplo, PI.

De acuerdo con el problema planteado, analiza si las constantes de matemáticas necesitan encapsulamiento o no

Escenario 6. Sistema de Registro de Calificaciones en una Escuela

Problema: En un sistema de registro de calificaciones escolares, se necesita rastrear las calificaciones de un estudiante en una asignatura.

De acuerdo con el problema planteado, analiza si una calificación necesita de encapsulamiento o no.

Escenario 7. Sistema de Configuración de Software

Problema: En un sistema de configuración de software, se almacenan opciones de configuración, como el idioma, que los usuarios pueden modificar según sus preferencias.
